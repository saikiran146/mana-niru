🚀 How to Start the Website
1

Add your Stripe API keys

Get your test keys from dashboard.stripe.com → API Keys
Set STRIPE_SECRET_KEY in server/.env
Set VITE_STRIPE_PUBLISHABLE_KEY in client/.env, then rebuild: 

cd client && npm run build
2

Start the server — that's it!

One terminal, one command. The server serves both the API and the website.

cd server && npm run dev
3

Click any link above to open the website

The status indicator at the top will turn green when the server is running. All links go to localhost:3001.
